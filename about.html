About Bundle Research

Independent research initiative led by a high school student in Saudi Arabia, focused on Earth observation, machine learning, and tools that make ambitious projects possible.

About me

I'm Mohamed Benkaddour, a high school student based in Riyadh, Saudi Arabia. I run "Bundle Research" as the umbrella name for my independent research and software work — currently focused on satellite-based environmental forecasting, with ongoing interests across machine learning, software engineering, and game development.

My approach is to take projects that interest me and push them toward publication-quality standards: leakage audits, reproducible code, peer-reviewable methodology. Bundle Research isn't a registered organization — it's a name for a body of work.

Interests

Earth observation Machine learning Remote sensing Climate & sustainability Software engineering Open source tools Game development C++ / Python / C# / Unity

Currently working on

  • Desertflow Global Bundle — per-region NDVI forecasting framework (active research).
  • Bundle Framework — open-source PySide6 GUI for running per-region ML experiments at scale.
  • Leakage-audit methodology — documenting the protocol that catches common failure modes in NDVI forecasting studies.
